# Vault use cases
Gathers ready to use use cases around vault solution

## Prerequisite
* Golang >= 1.16.6
* Vault >= v1.8.4

```
vault server --dev
export VAULT_ADDR='http://127.0.0.1:8200'
export VAULT_TOKEN="s.LdrzTRqiI7CIZpcSCCSTWpI4"
```

```
go build
./vault-usecase
```
